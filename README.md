Lightning Border fügt realistische Blitze hinzu, die sich nach und nach aufbauen und im Footer enden.

== Description ==

Dieses Plugin fügt Ihrer WordPress-Seite realistische Blitzeffekte hinzu, die sich von oben nach unten aufbauen und im Footer enden. Sie können die Blitzfarbe, Häufigkeit, den Abstand von der Mitte und die Anzahl der Blitze über eine benutzerfreundliche Einstellungsseite im WordPress-Dashboard anpassen.

== Installation ==

1. Laden Sie das Plugin-Verzeichnis in das `/wp-content/plugins/` Verzeichnis hoch.
2. Aktivieren Sie das Plugin über das 'Plugins' Menü in WordPress.
3. Passen Sie die Einstellungen unter 'Lightning Border' im WordPress-Dashboard an.

== Frequently Asked Questions ==

= Wie ändere ich die Farbe der Blitze? =

Gehen Sie zu den Einstellungen im WordPress-Dashboard und navigieren Sie zu "Lightning Border". Dort können Sie die Farbe der Blitze ändern.

= Kann ich die Häufigkeit der Blitze anpassen? =

Ja, Sie können die Häufigkeit der Blitze in den Einstellungen unter "Lightning Border" anpassen.
